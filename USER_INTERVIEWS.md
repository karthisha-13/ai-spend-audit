# User Interviews

## Interview 1
User: Student  
Problem: Uses multiple AI tools and doesn’t track cost  
Need: Simple way to check overspending  

## Interview 2
User: Freelancer  
Problem: Pays for tools but unsure of value  
Need: Cost vs usage insights  

## Interview 3
User: Team  
Problem: Shared accounts with no tracking  
Need: Team-level visibility  

## Insights
- Users want simple dashboards
- Users care about savings
- Clear suggestions are valuable
